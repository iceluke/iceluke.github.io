Chapter 2: An overview of the language

  • cout, cin: standard output/input streams

  • separate compilation

  • throw as “multilevel return” (try catch blocks)

  • user-defined types: using class, can define operations using operator{OPERAND}

  • destructor: ~{CLASSNAME}

  • abstract types: virtual (… =0), :public
